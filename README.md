# switch_multiple_fronts - Plugin for [Ampersand Remote](https://github.com/ralphConnors/AmpersandRemote).

### Description
Allows user to switch into multiple fronts at once.
It can also include switching to custom fronts as well.
### Usage
- Members fronting.
[![multiple-members.gif](https://i.postimg.cc/fbNzk7Hq/multiple-members.gif)](https://postimg.cc/TKtM4bvn)

- Custom fronts.
[![multiple-customs.gif](https://i.postimg.cc/br8VtrXJ/multiple-customs.gif)](https://postimg.cc/BLYCWqn9)

### How to implement
- Put the plugin into the `plugins` folder of Ampersand Remote
